express-simple-routing
======================


```
 GET     /          ->  actions.index
 GET     /new       ->  actions.new
 POST    /          ->  actions.create
 GET     /:Id       ->  actions.show
 GET     /:Id/edit  ->  actions.edit
 PUT     /:Id       ->  actions.update
 DELETE  /:Id       ->  actions.destroy
```
